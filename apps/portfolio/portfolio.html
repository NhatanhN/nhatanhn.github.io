Movie Browser

A mobile app that can track the movies you've watched or want to watch. Can also browse movie information from a database.

Project Demo ▶

Bring to a bigger screen by clicking on the button.

Technologies Used ▶

  • javascript
  • react native
  • OMDb's free API

Location Tracker

Pings your device's location 4 times a day and uploads it to a database. Automatically generated credentials can be used to view your data only.

Technologies Used ▶

  • javascript
  • react native
  • java
  • amazon web services
    • dynamodb (database)
    • lambda (server)

Note Taker

A website with a text editor that support image insertion. Allows you to take notes and save them to your account, as well export them to a .pdf file.

Technologies Used ▶

  • javascript
  • react
  • node.js
  • sqlite
  • json web tokens
